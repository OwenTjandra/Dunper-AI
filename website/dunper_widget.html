The Dunper chat popup

Drop this three-line snippet onto any website and your visitors get a Dunper AI chat bubble in the bottom-right corner — same conversation experience your customers see anywhere else, without leaving your site.

Embed code

Paste this just before </body> on every page you want the chat bubble on.

<!-- Dunper AI chat widget -->
<script>
  window.DUNPER_CHAT_URL = 'https://your-dunper-instance.example.com';
  window.DUNPER_LABEL    = 'Chat with us';
</script>
<script src="https://dunper.com/js/dunper-widget.js" defer></script>
Open the demo popup ↗ Copied

Configuration

All three globals are optional. The defaults work for a quick demo.

GlobalDefaultWhat it does
DUNPER_CHAT_URLdunper_chat.htmlURL the iframe loads inside the panel.
DUNPER_LABELChat with DunperTooltip + ARIA label on the bubble.
DUNPER_SUBTITLELive chat · we usually reply in secondsSmall text under the panel header title.
DUNPER_LOGO_URLimg/logo-nav.pngSmall logo shown in the panel header.

Trigger the popup from your own buttons

Any element on the host page with data-dunper-open or the class dunper-open opens the popup when clicked. Useful for "Talk to us" buttons inside the page content.

<button data-dunper-open>Talk to Dunper</button>

Notes for production

Use a permanent URL
Once Dunper has a permanent public chat URL (named tunnel or hosted domain), point DUNPER_CHAT_URL at it. The temporary Cloudflare tunnel rotates.

One bubble per page
The script is idempotent — loading it twice on the same page is a no-op.

Reduced motion
The bubble's pulse is disabled for users with prefers-reduced-motion: reduce.
